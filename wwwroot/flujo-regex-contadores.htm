1. Separacion tecnica de referencias

regex + limpieza + delimitacion

La primera capa limpia ruido de encabezado/pie y luego decide si el bloque contiene referencias APA, IEEE o mixtas. Si el corte no es confiable, se deja advertencia para revision manual.

1. NormalizarQuitar titulo de seccion, espacios y ruido repetido.
2. Localizar iniciosBuscar patrones APA e IEEE dentro del mismo bloque.
3. CortarOrdenar indices y extraer referencias individuales.
4. AdvertirMarcar mezcla o delimitacion incierta sin unir todo el bloque.
El diagrama puede desplazarse horizontalmente en pantallas pequenas.

2. Clasificacion y conteos

APA 7 estricta + agregaciones

Primero se decide el formato base. Solo si la referencia empieza como APA 7 entra a validacion por tipo. Los conteos globales usan solo referencias APA 7 correctas.

1. Formato baseAPA 7, IEEE, Vancouver, MLA, Chicago u otro estilo.
2. Tipo APAArticulo, libro, capitulo, ponencia, reporte o sitio web.
3. VeredictoCorrecta, con errores, invalida, otro formato o revision manual.
4. ConteoSolo entran años, titulos, autores y fuentes de APA 7 correctas.
Las flechas verdes terminan en datos que si se agregan a los conteos estrictos.

3. Revision del proceso contra el codigo

lectura tecnica
  • Correcto: el flujo empieza con extraccion de seccion, limpieza, normalizacion y separacion antes de analizar cada referencia.
  • Correcto: APA e IEEE se detectan por inicios distintos y despues se ordenan los indices para cortar referencias individuales.
  • Correcto: los formatos IEEE, Vancouver, MLA y Chicago no pasan por validacion APA campo por campo; se clasifican como otro formato.
  • Ajuste importante: los conteos de anio, titulo, primer autor y editorial/revista se calculan despues de filtrar referencias con IsApa7Compliant.
  • Ajuste importante: la editorial/revista no se cuenta desde texto bruto; se extrae desde ReferenceAnalysis porque depende del tipo detectado.

Apa7ReferenceService.cs - entrada del flujo 
public static List<string> FindReferences(string text)
{
    var section = ExtractReferenceSection(text);

    if (section.Length == 0)
    {
        return [];
    }

    var references = SplitReferences(section);
    return references.Count > 0 ? references : [section];
}
Apa7ReferenceService.cs - separacion individual 
private static List<string> SplitReferences(string section)
{
    var text = RemoveIeeeInterReferencePageNumbers(
        TextUtilities.NormalizeWhitespace(RemoveReferenceSectionTitle(section)));

    var starts = FindAllReferenceStartIndexes(text).ToList();

    var references = new List<string>();

    for (var index = 0; index < starts.Count; index++)
    {
        var start = starts[index];
        var end = index + 1 < starts.Count ? starts[index + 1] : text.Length;
        var reference = CleanReference(text[start..end]);

        if (reference.Length > 0)
        {
            references.Add(reference);
        }
    }

    return references;
}
Apa7ReferenceService.cs - APA + IEEE en el mismo corte 
private static List<int> FindAllReferenceStartIndexes(string text)
{
    var apaStarts = FindReferenceStartMatches(text)
        .Select(match => match.Index);
    var ieeeStarts = FindIeeeReferenceStartMatches(text)
        .Select(match => match.Index);

    return apaStarts
        .Concat(ieeeStarts)
        .Distinct()
        .Order()
        .ToList();
}
Apa7ReferenceService.cs - clasificacion de formato 
private static string DetectCitationStyle(string reference)
{
    if (IsIeeeReference(reference)) return "Formato IEEE";
    if (IsVancouverReference(reference)) return "Formato Vancouver";
    if (IsMlaReference(reference)) return "Formato MLA";
    if (IsChicagoReference(reference)) return "Formato Chicago";
    if (LooksLikeApa7(reference)) return "APA 7";

    return "Otro estilo";
}

Apa7ReferenceService.cs - conteo por anio 
public static SortedDictionary<int, int> CountYears(IEnumerable<string> references)
{
    var counts = new SortedDictionary<int, int>();

    foreach (var reference in references)
    {
        var match = Regex.Match(
            reference,
            @"\(\s*(?:\d{1,2}\s+de\s+[A-Za-zÁÉÍÓÚÜÑáéíóúüñ]+\s+de\s+)?(?<year>(?:19|20)\d{2})[a-z]?\s*\)",
            RegexOptions.CultureInvariant);

        if (!match.Success || !int.TryParse(match.Groups["year"].Value, out var year))
        {
            continue;
        }

        counts[year] = counts.TryGetValue(year, out var currentCount)
            ? currentCount + 1
            : 1;
    }

    return counts;
}
Apa7ReferenceService.cs - fuente sin romper dominios .com 
private static IEnumerable<string> SplitSentencePartsPreservingDomains(string text)
{
    var protectedText = Regex.Replace(
        text,
        @"\b[A-Za-z0-9-]+\.(?:com|org|net|edu|gov|mx|es|ar|cl|co|br)\b",
        match => match.Value.Replace('.', '\u001F'));

    return Regex.Split(protectedText, @"\.\s+")
        .Select(part => part.Replace('\u001F', '.'));
}

5. Regex clave

patrones tecnicos

Separacion de referencias

Inicio APA:
Apellido, I. (año).
Institucion. (año).

Inicio IEEE:
Apellido, I., ... (año).
Apellido, I.: ... (año).

Corte mixto:
detecta inicio APA + inicio IEEE
separar cada referencia por su propio inicio
